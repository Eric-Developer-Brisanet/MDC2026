Módulo 02

Lógica de Fatiamento (.iloc)

O fatiamento acontece através do método .iloc[]. Ele seleciona blocos de dados sem carregar todo o arquivo na memória RAM.

Formatos de Exportação:

  • CSV: Ideal para importação em bancos de dados.
  • XLSX: Padrão corporativo para relatórios.
  • TXT: Formato bruto tabulado para sistemas legados.

# Estrutura de Proteção

try:
    lote = df.iloc[i : i+5000]
    lote.to_csv(nome)
except Exception as e:
    print(f"Erro detectado: {e}")

import pandas as pd
import zipfile
import io

# Template Unibrisa
def brisa_fatiador_v1(caminho, linhas=5000):
    try:
        # Leitura Inteligente
        df = pd.read_excel(caminho) if caminho.endswith('.xlsx') else pd.read_csv(caminho)
        
        for i in range(0, len(df), linhas):
            lote = df.iloc[i : i + linhas]
            # Exportação de alta performance
            lote.to_csv(f"lote_unibrisa_{i//linhas + 1}.csv", index=False)
            print(f"Lote {i//linhas + 1} concluído.")
            
    except Exception as e:
        print(f"Erro fatal: {e}")

if __name__ == "__main__":
    brisa_fatiador_v1("seu_arquivo.xlsx")
